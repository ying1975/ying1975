\## 4. RUN\_ID Definition



RUN\_ID is a unique execution namespace.



Format:



YYYYMMDD\_HHMMSS



Example:



20260221\_113809



Purpose:



\- Prevent state collision

\- Support reproducibility

\- Enable independent report regeneration



RUN\_ID is auto-generated by run\_all.bat.  

Manual input is only required for report regeneration or debugging.



---



\## 5. Execution



Full pipeline:



run\_all.bat



Report only:



run\_report.bat <RUN\_ID> \[BT\_ROOT]



Default BT\_ROOT:



C:\\TradingSystem\\data\\out\\\_bt\_tmp

\## 7. Report Standards



Language: Traditional Chinese



Numeric Precision:



\- Summary metrics: 5 decimal places

\- Percentages: 5 decimals

\- Turnover: 5 decimals

\- Trade value: integer with comma



PDF:



\- Landscape A4

\- 10 columns

\- Auto wrap name column

\- Chinese font fallback supported

\## 8. Batch Encoding Rule



All .bat files must be saved as:



\- ANSI  

or  

\- UTF-8 without BOM  



Never use UTF-16 or UTF-8 with BOM.
